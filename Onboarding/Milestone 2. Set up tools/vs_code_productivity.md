### VS Code productivity



###### Put in your top 10 shortcuts.



1. Ctrl + P: type and search for filename and open file. Does not need to be an automatic match, can only type the first few letters of filename
2. Crtl + G: Go to line number
3. Alt + left/right: navigate backward and forward. Will be easier jumping between files
4. Ctrl + D: Select next occurrence. Select a word, and press Ctrl+D for selecting next match 
5. Alt + Click: Adding cursor to where my click is.
6. Ctrl + shift + L: Select all occurrences. Select a word, the use shortcut to select all matching words.
7. Ctrl +/-: toggle line comment. can comment and uncomment lines easily
8. Alt + up/Down: move line up and down
9. Ctrl + shift + K: Delete current line
10. Ctrl + ': switching between code and terminal
